# 🚀 Featured Projects

## ✈️ AeroMotion AI

**Gesture-controlled desktop automation suite** using Computer Vision
and MediaPipe that enables touchless interaction through a Virtual
Mouse, Magic Canvas, and Hand Gesture Controller.

**Tech:** Python • OpenCV • MediaPipe • PyAutoGUI • CustomTkinter •
NumPy

🔗 https://github.com/Sakshi-Khutwad/AeroMotion-AI

------------------------------------------------------------------------

## 🎥 AI YouTube Video Downloader & Summarizer

Downloads YouTube videos/audio and generates structured AI-powered
summaries using transcription and Gemini, complete with timestamps.

**Tech:** Node.js • Express • AssemblyAI • Gemini API • FFmpeg •
ytdl-core

🔗 https://github.com/Sakshi-Khutwad/yt-video-downloader-summarizer

------------------------------------------------------------------------

## 🤟 SignVision

End-to-end **English to Indian Sign Language (ISL)** conversion platform
combining NLP, semantic search, speech recognition, and a React Native
mobile application.

**Tech:** FastAPI • React Native • Expo • SpaCy • ChromaDB • OpenAI API

🔗 https://github.com/Sakshi-Khutwad/SignVision

------------------------------------------------------------------------

## 🧠 English → ASL Translation

Sequence-to-Sequence LSTM model translating English sentences into
**American Sign Language Gloss** to improve accessibility for the deaf
community.

**Tech:** TensorFlow • Keras • Streamlit • Pandas • NumPy • Scikit-learn

🔗 https://github.com/Sakshi-Khutwad/english-to-asl-language-translation

------------------------------------------------------------------------

## 🧳 TravelOpti

Travel route optimizer based on the **Travelling Salesman Problem**,
combining route optimization algorithms with an interactive React
interface.

**Tech:** React • Vite • Tailwind CSS • Python • OSRM • Nominatim

🔗 https://github.com/Sakshi-Khutwad/TravelOpti

------------------------------------------------------------------------

## 🏥 MediBot

AI-powered medical report analyzer that extracts text from images and
PDFs using OCR and explains reports in simple language with Gemini AI.

**Tech:** Streamlit • OpenCV • Tesseract OCR • Gemini API • Camelot

🔗 https://github.com/Sakshi-Khutwad/MediBot
